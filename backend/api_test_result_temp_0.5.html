Anmeldelse av Skid Row si "Subhuman Race", ei plate som meir eller mindre var slutten på bandet som ei stormakt i rocken. Ei plate der dei prøvde å etterape grungen som hadde blitt så stor, men som viste seg å være ei sjanger dei ikkje meistra.

Skid Row, ein gong i tida ein tungvektar i rocken, prøvde å kaste seg på grungetoget med "Subhuman Race". Men det viste seg raskt at dei hadde trøbbel med å følgje den nye bølgja. Som ein fisk på land, vrengte dei seg desperat for å tilpasse seg, men det var som å sjå ein elefant prøve å balansere på ein tynn gren - det gjekk ikkje bra.

Bandet som ein gong hadde potensiale til å vera stort, enda opp med ei plate som var som ein dårlig kopi av noko dei aldri kunne forstå heilt. Det var som å sjå ein amatørtryllekunstnar prøve å imponere med triks dei ikkje hadde øvd nok på. Skid Row kasta seg ut i grungeverda med "Subhuman Race", men det var som å kaste perler for svin.

Med låtar som prøvde å vere råe og autentiske, enda dei berre opp med å høyrast ut som ein parodi av seg sjølv. Det var som å sjå ein skodespelar som gløymer replikkane sine midt på scena - pinleg og uverdig. Skid Row si "Subhuman Race" blei ikkje berre ei skive, det blei ei påminning om korleis ein kan mislukkast når ein prøver for hardt utan å forstå kva ein eigentleg held på med.

Det er som om Skid Row gjekk ut i ein storm utan paraply, og enda opp våte, forfrosne og flaue. "Subhuman Race" er ikkje berre ein plate, det er ei tragedie - ein påminning om kor skjør suksess kan vere, og kor fort ein kan falle når ein mistolkar trendar og publikum.
