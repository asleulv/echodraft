Det var ein gong eit band som rådde over rockeverda. Skid Row, med Sebastian Bach i spissen, hadde alt som skulle til for å setje fyr på 80- og tidleg 90-talet med sin blanding av glam og hard rock. Men så kom grungen sigande inn, som ei flodbølgje som skulle vaske bort det meste av det som hadde smakt av hairspray og lêrjakker. Midt i denne sjangerstormen prøvde Skid Row å finne sin plass på nytt med albumet Subhuman Race. Men istadenfor å bli hyllet som ein ny start, vart albumet meir eller mindre slutten på Skid Row som ein stormakt i rocken.

Med Subhuman Race gjorde Skid Row eit forsøk på å etterape grungen, ei sjanger som stod i sterk kontrast til bandet sitt opphavlege uttrykk. Dette var eit modig, men kanskje også eit feilrekna steg. For sjølv om grungen verkeleg regjerte, så var det eit musikalsk landskap som krevde autentisitet. Det var som om Skid Row prøvde å snakke eit språk dei ikkje heilt meistra, og resultatet føltes forvirra og utilfredsstillande for mange av deira faste tilhengjarar.

Albumet bød på tunga, mørkare riff og ein meir nedpå stemning enn det fansen var vant med. Men i staden for å bli sett på som ein evolusjon, ble det meir oppfatta som ein identitetskrise. Låtar som "My Enemy" og tittelsporet "Subhuman Race" viste potensiale, men fanga ikkje heilt den rå, uforsonlege energien som definerte det beste innan grunge. Det var som om Skid Row stod med eit bein i glamrocken og det andre forsiktig dippa i grunge-vatnet, utan å vere heilt sikre på kor dei ville vere.

Det var ikkje at Subhuman Race var utan sine høgdepunkt. Musikalsk sett, var det tekniske ferdigheiter på display, og Sebastian Bach si stemme hadde framleis den kraften og rekkevidda som kunne gi deg gåsehud. Men albumet mangla den umiddelbare tiltrekningskrafta og sjarmen frå tidlegare suksessar som "Slave to the Grind". Det føltes som om Skid Row prøvde for hardt å tilpasse seg ein trend, i staden for å følgje sin eigen musikalske veg.

I ettertid kan ein sjå på Subhuman Race som eit dokument over ein tid der musikkbransjen var i rivande utvikling, og mange band frå 80-tallet sitt glamrock-scene kjempa for relevans. For nokre var det starten på noko nytt, for andre, som Skid Row, blei det ein bauta over det som ein gong var. Albumet står som eit vitnesbyrd om bandet sin vilje til å eksperimentere og utfordre seg sjølve, sjølv om det ikkje heilt klarte å lande blant stjernene.

Til syvande og sist kan ein ikkje anna enn å føle eit visst nivå av nostalgi og sympati for Skid Row sitt Subhuman Race. Det representerer ein viktig, om enn vanskeleg, periode i rockehistoria. Og sjølv om det kanskje ikkje er bandet sitt sterkaste verk, så er det ein viktig del av deira historie - eit ærleg forsøk på å navigere i musikkens skiftande tidvatn.

Det er lett å kritisere Skid Row for å ha mista sin veg med Subhuman Race, men det er òg viktig å anerkjenne motet det tok å prøve noko nytt. I retrospekt kan vi kanskje sjå albumet som eit misforstått meisterverk som ikkje heilt fann sitt publikum i sitt øyeblikk. Eller kanskje som ein påminnar om at ikkje alle eksperiment er dømt til å lukkast, men at det å strekke seg mot nye horisontar alltid er ein risiko verdt å ta.
