Anmeldelse av Skid Row si Subhuman Race er eit dykk ned i ei tid der bandet var på veg ut frå rampelyset, med ein plate som meir eller mindre markerte slutten på deira storheitstid innan rocken. På denne plata prøvde dei å omfavne grungebølgja som på det tidspunktet herja som ein tsunami i musikkverda, men det viste seg å vere ei sjanger dei ikkje klarte å meistre.

Skid Row hadde tidlegare levert hits som "Youth Gone Wild" og "18 and Life" som sette dei på kartet som ein av rockens fremste. Men med Subhuman Race gjorde dei eit forsøk på å tilpasse seg den nye musikktrenden, og det blei tydeleg at det ikkje var noko dei fekk til like godt.

Plata bar preg av ein desperat streben etter å henge med i tida, men samtidig mangla dei autentisiteten og sjela som gjorde at grungebanda virkelig resonerte med publikum. Det vart eit forsøk på å kopiere noko som allereie var unikt, og resultatet blei ein samling låtar som ikkje klarte å finne sin eigen identitet.

Skid Row si Subhuman Race er difor ein påminning om kor skjør suksess kan vere i musikkbransjen. Å prøve å etterape suksessen til andre utan å tilføre noko eigenart kan ende opp med å bli deira bane. Bandet som ein gong regjerte rockescenen, enda opp med å slå feil steg for steg på denne plata, og det vart tydeleg at det var ei æra som var på veg mot slutten.
