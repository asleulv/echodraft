Anmeldelse av Skid Row si Subhuman Race, ei plate som meir eller mindre var slutten på bandet som ei stormakt i rocken. Ei plate der dei prøvde å etterape grungen som hadde blitt så stor, men som viste seg å være ei sjanger dei ikkje meistra.

Eg har ein ting å seie om Skid Row si Subhuman Race - det er som å sjå ein gammal rockestjerne prøve å klemme seg inn i trange, slitte skinnbukser frå 80-talet. Det fungerer berre ikkje. Skid Row, ein gong ein stolt deltakar i rockens hall of fame, leverer her ei plate som ber preg av desperasjon og ein kamp for å halde seg relevante i ein tid då musikken hadde allereie bevega seg vidare.

Bandet, som ein gong hadde potensialet til å bli noko stort, endar opp med å klinge som ein dårlig kopi av seg sjølv - ein parodi på den suksessen dei ein gong hadde. Subhuman Race er som ein mislykka karaokekveld der vokalisten prøver å imponere med høge tonar, men berre endar opp med å få publikum til å riste på hovudet i forakt.

Det er som om Skid Row prøvde å henge med på den nye bølgja av grunge som herja på den tida, men i staden for å omfavne det autentiske og råe ved sjangeren, endar dei opp med ei vass og tam etterlikning som manglar sjel og substans. Det er som å sjå ein dinosaur forsøke å danse ballett - det er berre ikkje naturleg.

Subhuman Race er ei plate som prøver så hardt å være noko det ikkje er, at det berre blir pinleg. Det er som å sjå ein middelaldrande mann prøve å vere ung og trendy ved å kle seg i klede som er to nummer for små. Det fungerer berre ikkje, og det blir berre flaut å sjå på.

Skid Row si Subhuman Race er eit trist kapittel i bandet si historie, ein påminning om korleis ein gong stolte rockehelter kan falle så djupt. Det er som å sjå ein ein gong stolt løve bli redusert til ein ynkeleg kattunge som mjauar etter oppmerksomheit.
